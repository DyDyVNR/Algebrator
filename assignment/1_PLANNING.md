## Title: Algebrator

### Project Description

This program is a tool that serves students and educators in Lineaer ALgebra Course. It computes matric operations such as addition, subtraction, multiplication, inversion, tranposition, and Guassian Elimination. 

### Project Selection

I chose this project because I can demonstrate what we are going to learn in class by developing this program. Although on the surface it sounds like a basic calculator, it's actually a perfect showcase of computer science and mathematic concepts. Apart from that, it can be used by students who struggle with the abstract concepts in linear algebra.
